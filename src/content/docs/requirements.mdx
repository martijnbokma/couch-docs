To use Couch you must at least have a working knowledge of HTML and CSS and know the basics of creating a website. You should also be familiar with the use FTP to upload files to your server. No previous knowledge of PHP is required.

### Server Requirements

- **PHP**: 5.0.0 or greater
- **MySQL**: 4.1.2 or greater
- **Webserver**: Apache (or compatible Web Servers like LiteSpeed etc.)

:::danger[Important]
Microsoft IIS web server is NOT supported.
:::

- (Optional) GD library compiled into PHP for CAPTCHA.
- (Optional) Apache mod_rewrite module for pretty URLs.

### Local Development Options

#### Windows
- **XAMPP**: Complete package with Apache, MySQL, and PHP ([Download](https://www.apachefriends.org/download.html))
- **WampServer**: Alternative to XAMPP ([Download](https://www.wampserver.com/en/))
- **MAMP**: Professional version available for Windows ([Download](https://www.mamp.info/en/downloads/))

#### macOS
- **MAMP**: Popular choice for Mac users ([Download](https://www.mamp.info/en/downloads/))
- **XAMPP**: Also available for macOS ([Download](https://www.apachefriends.org/download.html))
- **Homebrew**: Install Apache, MySQL, and PHP separately ([Install Guide](https://brew.sh/))
- **Docker**: Run a complete LAMP stack in containers ([Download](https://www.docker.com/products/docker-desktop))

:::note
For local development, any of these options will work fine. Choose the one that best fits your workflow and experience level.
:::
