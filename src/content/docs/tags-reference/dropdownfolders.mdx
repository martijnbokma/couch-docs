The `dropdownfolders` tag creates an HTML dropdown list of the folders belonging to a template.

```php title="Basic Dropdown Example"
<cms:dropdownfolders masterpage="news.php" />
```

## Parameters

Supports the same parameters as the `folders` tag, plus one additional parameter - `show_count`.

- `masterpage`: Template to list folders from
- `root`: Root folder to start listing from
- `childof`: Show only children of specified folder
- `hierarchical`: Show folders in hierarchical order
- `depth`: Maximum depth of folders to show
- `orderby`: Field to sort folders by
- `order`: Sort order (`asc`/`desc`)
- `exclude`: Folders to exclude
- `extended_info`: Include additional folder information
- `show_count`: Show number of items in each folder

Please see the [**folders**](../folders) documentation for an explanation of these parameters.

## Variables

This tag is self-closing and does not set any variables of its own.
