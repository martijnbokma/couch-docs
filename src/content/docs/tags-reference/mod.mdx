The `mod` tag accepts two parameters and returns the remainder obtained after dividing the first parameter by the second.
The parameters may be literal strings, variables or output of other tags.

```php title="Basic Modulo Example"
<cms:set my_count="23" />
<cms:mod my_count "5" />
```

The snippet above outputs `3`, which is what remains after dividing 23 by 5.

## Parameters

The tag expects two parameters as the operands:
- First parameter: The dividend (number to be divided)
- Second parameter: The divisor (number to divide by)

## Variables

This tag is self-closing and does not set any variables of its own.
